Detailed Description

Fichier source pour la gestion d'un tas (max heap) utilisé dans les opérations sur les quadtree.

Ce fichier contient les définitions des fonctions pour créer, insérer, extraire et libérer un tas max utilisé pour optimiser les opérations de subdivision dans les quadtree.
